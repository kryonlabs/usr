Rio - Plan 9 Window Manager in Go

Rio is a reimplementation of the 9front rio window manager in Go. It provides window management, event handling, and a 9P filesystem interface for the TaijiOS desktop environment.

Architecture

User Applications (Acme, sam, Kry UI)
              ↓ 9P2000 Protocol
           Rio (Window Manager)
              ↓ Draw Protocol
           9xe (Execution Layer)
              ↓
           SDL2 Graphics

Features

  • Window Management: Create, destroy, move, resize, hide/show windows with stacking order
  • Event Handling: Mouse and keyboard event processing and routing
  • 9P Filesystem: Complete control via /mnt/wm filesystem interface
  • Draw Bridge: Legacy Plan 9 apps (acme, sam) run unmodified via /mnt/wm/win/[id]/draw
  • Menu System: 9front-style right-click menu for window operations
  • Kry Integration: Kry UI toolkit extends rio windows with modern widgets

Filesystem Interface

/mnt/wm/
├── ctl                   # Global control
├── new                   # Create new window
└── win/[id]/             # Window directories
    ├── ctl               # Window control
    ├── title             # Window title (read/write)
    ├── rect              # Geometry "x y w h"
    ├── visible           # "1" or "0"
    ├── draw              # 9P draw protocol (legacy apps)
    ├── event             # Event stream
    └── kry/              # Kry UI extensions
        ├── type          # Window type
        └── wg/           # Widget directories
            └── [name]/    # Widget control files

Window Types

  • LegacyWindow: Direct draw access for Plan 9 apps
  • KryWindow: Modern UI with Kry toolkit widgets
  • HybridWindow: Both draw surface and Kry UI elements

Usage

# Build rio
cd rio
go build -o bin/rio ./cmd/rio

# Run rio
./bin/rio

# Create window via 9P
echo "640 480" > /mnt/wm/new

# Control window
echo "My Title" > /mnt/wm/win/1/title
echo "100 100 740 580" > /mnt/wm/win/1/rect
echo "1" > /mnt/wm/win/1/visible

API

import "github.com/kryonlabs/rio/lib/rio"

// Create window manager
wm := rio.NewWindowManager(screen)

// Create window
win, err := wm.CreateWindow(rect, "My Window")

// Window operations
wm.TopWindow(win.ID)
wm.MoveWindow(win.ID, 100, 100)
wm.ResizeWindow(win.ID, 640, 480)

Dependencies

  • 9xe - Draw protocol and graphics backend
  • go-ninep - 9P protocol implementation
