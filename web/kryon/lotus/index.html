Lotus

Lotus is a personal operating system for Varvara with a zen-like focus. A fork of the potato desktop environment, adding custom utilities for meditation and mindfulness.

Applications

  • Inbe - Guided Wim Hof breathing method meditation app

About Varvara

Varvara is a virtual computer built on the UXN virtual machine. It provides a consistent platform for running portable graphical applications across different operating systems.
