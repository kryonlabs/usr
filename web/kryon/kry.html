Kry - Plan 9 UI Toolkit

Kry is a filesystem-based UI toolkit that extends Rio windows with 88 different widget types. It provides a declarative UI syntax and complete control via 9P filesystem operations.

Architecture

Kry (UI Toolkit)
  - 88 widget types
  - Layout managers
  - Event handling
      ↓ Imports
   Rio (Window Manager)
      ↓ Draw Protocol
   9xe (Execution Layer)

Widget Types (88 Total)

  • Input (24): text, password, checkbox, radio, dropdown, slider, etc.
  • Buttons (8): button, submit, icon, FAB, segmented, etc.
  • Display (11): label, heading, image, progress bar, badge, etc.
  • Containers (11): container, box, frame, scroll, split, tab, etc.
  • Lists (8): list item, tile, view, grid, tree, chip, tag
  • Navigation (8): menu bar, toolbar, nav bar, breadcrumb, etc.
  • Menus (4): dropdown, context, cascading, popup
  • Forms (4): form, group, validation, input group
  • Windows (4): window, dialog, scaffold, fullscreen
  • Specialized (8): canvas, SVG, chart, color picker, calendar, etc.

Declarative UI

window "My Form" {
    rect: 100 100 400 300

    label "Name:" {
        rect: 10 10 380 30
    }

    text-input nameInput {
        rect: 10 40 380 60
        placeholder: "Enter your name"
    }

    button submitBtn {
        rect: 10 70 100 90
        text: "Submit"
    }
}

Filesystem Interface

/mnt/wm/win/[id]/kry/
├── type              # Window type
└── wg/               # Widget directories
    ├── nameInput/
    │   ├── type      # "text-input"
    │   ├── text      # Current value
    │   ├── value     # Widget state
    │   └── rect      # "10 40 380 60"
    └── submitBtn/
        ├── type      # "button"
        ├── text      # "Submit"
        └── rect      # "10 70 100 90"

Layout System

  • Row Layout: Horizontal arrangement with gaps
  • Column Layout: Vertical arrangement with gaps
  • Grid Layout: 2D grid with configurable columns
  • Absolute Layout: Manual positioning

Usage

import "github.com/kryonlabs/kry/lib/kry"

// Create Kry window
kw, err := kry.CreateKryWindow(wm, "My Window")

// Add widgets
label, _ := kw.AddWidget(kry.LabelWidget, "nameLabel")
label.SetProperty("text", "Name:")
label.SetProperty("rect", "10 10 380 30")

input, _ := kw.AddWidget(kry.TextInputWidget, "nameInput")
input.SetProperty("placeholder", "Enter your name")

button, _ := kw.AddWidget(kry.ButtonWidget, "submitBtn")
button.SetProperty("text", "Submit")
button.OnClick(func() {
    value := input.GetProperty("value")
    fmt.Println("Submitted:", value)
})

Dependencies

  • Rio - Window manager (base)
  • 9xe - Draw protocol and graphics
