What is Taiji OS?

TaijiOS is a Plan 9-based operating system built with Go, featuring a modular architecture with 9xe (Plan 9 execution layer), rio (window manager), and kry (UI toolkit). It provides a modern implementation of Plan 9 principles with 9P filesystem interfaces throughout.

Architecture

  • 9xe: Plan 9 binary execution layer with SDL2 graphics backend. Runs Plan 9 a.out binaries via CPU emulation.
  • rio: Go-based window manager reimplementing 9front rio. Provides window management, event handling, and 9P filesystem interface.
  • kry: UI toolkit extending rio with 88 widget types. Filesystem-based API for declarative UI creation.
  • 9P Protocol: Every resource (windows, widgets, events) is exposed via 9P filesystem at /mnt/wm.
  • Legacy App Support: Draw file bridge allows apps like acme to run unmodified via 9P draw protocol.
  • Modular Design: Each component is an independent Go module that can be used standalone or combined.

Quick Start

# Clone the repository
$ git clone https://github.com/KryonLabs/TaijiOS.git
$ cd TaijiOS

# Initialize submodules
$ git submodule update --init --recursive

# Build all components
$ make all

# Run rio window manager
$ make dev-rio
